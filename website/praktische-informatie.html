Praktische informatie

Voorkomen verrassingen, lees voor de run de onderstaande informatie goed door.

Start-/finishlocatie

De start-/finishlocatie is in park Brakkenstein, Nijmegen. Hier kan je je aanmelden, en hier bevindt zich de shirtuitgifte en een bewaakte tassenopslag. Parkeren is gratis en kan op de hieronder aangegeven parkeerplaats. Voor navigatie: gebruik het adres Heyendaalseweg 141, Nijmegen. In het park zijn ook fietsenstallingen beschikbaar.

Starttijden

De starttijden zijn verdeeld over de dag tussen ongeveer 10:00u en 14:30u. De starttijden worden in de week voor de run gepubliceerd. De 8 km zal eerst starten vervolgens zal de 5 km individueel starten en als laatst start 5 km groep.

Prijsuitreiking

De prijsuitreiking voor de 8km ind. vindt plaats om 13:30. De prijsuitreiking voor de 5km (ind. en groep) vindt plaats om 15:45.

Omkleden en douchen

Het omkleden vooraf kan in de omkleedtenten op het start-/finishterrein. Douches zijn niet aanwezig op het start-/finishterrein, maar je kan gebruik maken van de douches op het Radboud Sportcentrum. Zie daarvoor de kaart hiernaast, en volg op de dag zelf de bewegwijzering.

Het deelnemersshirt

Aan het begin van de run krijg je je deelnemersshirt. Het is verplicht dit shirt te dragen tijdens de run zodat je herkenbaar bent. Daarnaast heb je een mooi sportshirt die je later nog kan gebruiken!

Het polsbandje

Aan het begin van de run krijg je een polsbandje. Dit is je trots en je eer, deze moet je koste wat kost zien te behouden. Dit kan echter alleen door alle hindernissen succesvol te overwinnen. Lukt een hindernis niet, dan mag je het in principe zo vaak opnieuw proberen als je wilt. Als je opgeeft zal de jurypost je bandje doorknippen. Je krijgt dan twee uur straftijd.

De route

De route is aangegeven met rood/wit lint. Wees tijdens de run oplettend zodat je geen afslagen mist; dit kan diskwalificatie tot gevolg hebben.

De hindernissen

Bij iedere hindernis staat een jurypost die je uitlegt hoe je de hindernis moet nemen en die controleert of de uitvoering correct is. De instructies van de jury zijn altijd leidend. Blijf alsjeblieft vriendelijk tegen de jury, zij doen hun werk vrijwillig, zonder hen was de run niet mogelijk geweest.

Verversingspost

Onderweg kom je een verversingspost tegen, waar je weer op kracht kan komen met wat eten en drinken. Zo hebje weer genoeg energie om de rest van het parcours te overwinnen.

Kleding

Een survivalrun gaat grotendeels over onverhard terrein. Trek dus degelijke hardloopschoenen aan met een goed profiel. Trailrunschoenen zijn het allerbeste. Daarnaast zul je door hindernissen klimmen en langs touwen schuren, dus is het raadzaam om een lange hardlooptight te dragen. Ben je eigenwijs en draag je toch een korte broek, zorg dan voor lange dikke sokken. Het wedstrijdshirt krijg je van ons, hierop staat je startnummer. Zorg ervoor dat deze tijdens de run altijd zichtbaar is. Tijdens de run wordt je nat, dus vergeet niet om een extra set droge kleren mee te nemen voor na de run.

Wees op tijd

Zorg ervoor dat je ongeveer een uur voor je start aanwezig bent op het start-/finishterrein. Hier kun je je wedstrijdshirt en polsband ophalen, en een warming-up doen. 5 minuten voor je start moet je aanwezig zijn in het startvak.

Pre-run clinic

Voorafgaand aan de run zal de pre-run clinic worden georganiseerd. Dit is een survivalruntraining van 1,5 uur waarbij je wordt klaargestoomd voor je eerste survivalrun. Hier leren wij je de basistechnieken van survivalrun: de voetklem, de swingover, de apenhang en meer. Klik hier voor meer info.

Voorwaarden

Voor deelname aan de SurviFELrun dien je akkoord te gaan met het Algemeen Regelement en het Deelnemersreglement van de Survivalrun Bond Nederland.
